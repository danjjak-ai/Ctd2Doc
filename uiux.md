# UI/UX: Streamlit-Based Pipeline Dashboard

구글 Colab 환경에서 `Streamlit` 또는 `Gradio`를 로컬 터널링(`pyngrok` 혹은 `localtunnel`)으로 띄워 실시간으로 자율 학습 상태를 모니터링하는 대시보드 화면 명세입니다.

## 1. 컴포넌트 레이아웃 구성

### 상단부: 시스템 상태 요약 타일 (Summary Cards)
- **현재 학습 반복 회차 (Iteration):** `3회차`
- **현재 활성화된 모델 가중치:** `gemma-2-9b-qlora-iter-3`
- **평균 파이프라인 스코어:** `78.5 / 100` (목표치: 85)
- **사용 중인 VRAM 상태:** `32.4 GB / 40.0 GB (A100)`

### 중앙부: 실시간 스코어 추이 그래프 (Line Chart)
- **X축:** 의약품 개수 / 학습 횟수 흐름
- **Y축:** Judge Agent가 내린 종합 점수 (0 ~ 100)
- **시각화 요소:** 목표선(85점)에 도달하는 과정을 꺾은선 그래프로 표기.

### 하단부 좌우 분할 스크린 (Side-by-Side Comparator)
- **좌측 패널 (Ground Truth):** 원본 의약품 인터뷰 폼 / 쿠스리노 시오리 텍스트 데이터.
- **우측 패널 (Generated Output):** 현재 회차의 Gemma 모델이 생성해 낸 출력 데이터.
- **차이점 하이라이트 (Diff Viewer):** 정답 문서 대비 누락된 임상 통계 수치나 잘못 번역된 용어는 **빨간색 하이라이트** 처리.

## 2. 사용자 인터랙션 요소
- **[자율 모드 시작/정지] 버튼:** 사람이 개입하지 않고 약품 리스트 전체를 돌며 무한 루프 학습을 수행할지 제어.
- **[수동 체크포인트 저장] 버튼:** 현재 점수가 만족스러울 경우 허깅페이스 허브(Hugging Face Hub)나 구글 드라이브에 LoRA 어댑터 가중치를 수동 업로드.
